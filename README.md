# HyperWallet

HyperWallet is a multi-chain cryptocurrency wallet that supports Bitcoin, Ethereum, Solana, Monero, Zcash, and more.

## Download

Download the latest release for your platform:

- **macOS**: [Download DMG](https://github.com/kylebolton/HyperwalletPublic/releases/latest)
- **Windows**: [Download Installer](https://github.com/kylebolton/HyperwalletPublic/releases/latest)
- **Linux**: [Download AppImage or DEB](https://github.com/kylebolton/HyperwalletPublic/releases/latest)

Visit the [Releases page](https://github.com/kylebolton/HyperwalletPublic/releases) for all available downloads.

## Features

- Multi-chain support (Bitcoin, Ethereum, Solana, Monero, Zcash)
- Secure wallet management
- Cross-platform (macOS, Windows, Linux)
- Automatic updates

## Auto-Updates

HyperWallet automatically checks for updates and notifies you when new versions are available. Updates are downloaded and installed automatically.
